About

With this tool you can create your own bussiness card.

How to use

Complete the form, then submit. If you are redirected to the same page try to enter other texts or to upload other photos. You can use PNG or JPG files. Note that PNG are higher quality. Also hover over the input labels to learn about input limits.

Expected output

If everything is done right you should be redirected to a page where is the final image. Here you can download it. The image is resized 340x189 pixels, that is roughly 9x5 centimeters(340.16x188.98 pixels).

Afterwards you can print your bussiness card.
